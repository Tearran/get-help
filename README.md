# Self Help
self-help is a bash script example of get ops flags and 'source && exicution'

## Features 
- Chose any filename
- getops example
- example to list functions by placing ##:: in the function

$ `self-help -h`
```bash

Usage: self-help [ -h ] [ --help ]
Options:
        -w      Run Self-Help CLI-UI
        -h      Print this help.
        -v      Print version info.

```

$ `self-help -w`
